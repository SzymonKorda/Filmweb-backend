# Movies server-side application
Backend application for managing database of movies and actors. 



API documentation:
https://filmweb.skorda.pl/swagger-ui/#

Used technologies: 
- Java 11,
- Spring Boot, 
- Hibernate, 
- REST API, 
- Swagger
- Pagination 
- jUnit 5, Mockito,
- Maven,
- JWT, 
- Project Lombok,
- Docker,
- PostgreSQL
